# Filosofía

NekoTools es una colección de herramientas **rápidas, privadas y modernas** creadas para desarrolladores, diseñadores y creadores de contenido.

Nuestro enfoque se basa en tres pilares fundamentales:

## 1. Privacidad por defecto

La mayoría de las herramientas funcionan 100% en el navegador (Client-Side).

No almacenamos datos. No requerimos registro. No rastreamos usuarios.

El usuario mantiene control total sobre sus archivos e información.

## 2. Velocidad y simplicidad

Cada herramienta resuelve un problema específico de forma directa y sin fricciones.

El objetivo es obtener un resultado en segundos, sin configuraciones complejas ni procesos innecesarios.

## 3. Calidad sobre cantidad

Preferimos construir menos herramientas pero con un nivel de acabado profesional, utilizando tecnologías modernas y patrones de diseño actuales.

---

# Principios

## Prioridad: Experiencia del usuario

La herramienta está primero, la marca es un complemento.

Neko (la mascota) acompaña la interfaz pero nunca compite con la funcionalidad.

## Decisiones técnicas

- Siempre que sea posible → Procesamiento local (WebAssembly, WebGPU, Transformers.js, ONNX)
- Solo usar backend cuando no exista alternativa viable
- Carga rápida: objetivo < 500ms
- Sin dependencias pesadas ni frameworks innecesarios

## Diseño moderno

- Estética inspirada en Linear, Raycast, Vercel, Arc Browser
- Modo oscuro como estándar
- Componentes consistentes
- Animaciones sutiles y pulidas

## Comunicación

- Directa y clara
- Técnica pero accesible
- Enfocada en resolver problemas reales

# Balance de Marca

NekoTools mantiene una proporción visual de:

- **80% Producto profesional**
- **20% Personalidad Neko**

Debe sentirse como una herramienta confiable utilizada diariamente por profesionales, pero con detalles que la hacen memorable.

---

# Ecosistema

NekoTools forma parte de **CinloDev** como:

- Línea de herramientas web
- Enfoque en experiencia frontend moderna
- Construcción de utilidades reutilizables

Mientras otras áreas de CinloDev resuelven problemas específicos, NekoTools se enfoca en crear un ecosistema de soluciones universales para la comunidad digital.
